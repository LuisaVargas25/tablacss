Css

Definición de CSS

Css es un lenguaje diseñado pra el control de la apariencia y presentación de las páginas web. Permite definir los estilos como color, fuente, margenes, tamaño y animaciones, mejora la experiencia visual que tiene el usuario.

La Importacia de CSS en el Desarrollo Web

Facilita la gestión del código al mantener HTML (que es la estructura) y CSS (el diseño) separados. También permite adaptar la apariencia del sitio web en diferentes dispositivos.

 

JavaScript

La Importacia de JavaScript en la Web
JavaScript lenguaje de programación que permite agregar la interactividad y el dinamismo en las páginas web.
Interactivida y Dinamismo:Este sirve para que los sitios web sean más atractivos con animaciones y efectos.
Validación de Formularios: Reduce la carga al servidor al validar los datos en el navegador antes de enviarlos.
Manipulación del Dom: Permite cambiar el contenido, la estructura y el estilo de una página sin necesidad de recargarla.
